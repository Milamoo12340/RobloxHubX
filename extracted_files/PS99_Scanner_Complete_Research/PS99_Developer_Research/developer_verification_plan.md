# Pet Simulator 99 Developer Verification Plan

## Primary Sources to Check
1. Official BIG Games Roblox group
2. Big Games Pets group
3. Big Games Experimental group
4. Big Games Super Fun group
5. Developer profiles connected to Preston (BIG Games founder)

## Verification Methods
1. Check if developer is listed in official BIG Games groups
2. Verify if they're friends with known PS99 developers
3. Look for official roles or mentions of PS99 development in their profile
4. Check if they have uploaded assets that match PS99 style/naming
5. Cross-reference with developer credits or announcements

## Documentation Format
For each verified developer:
- Roblox User ID
- Username
- Role (if known: developer, tester, builder, etc.)
- Verification sources (which methods above confirmed they're legitimate)
- Link to Roblox profile

## Developer Asset Scan Parameters
For each verified developer:
- Asset creation date range (focus on recent 2-3 months)
- Asset types to scan (models, images, meshes, etc.)
- Keywords to match ("PS99", "Egypt", "Anubis", "Update", etc.)
